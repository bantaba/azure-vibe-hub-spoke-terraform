# Documentation

This directory contains comprehensive documentation for the Terraform security enhancement project.

## Structure

- `security/` - Security-related documentation
- `setup/` - Setup and configuration guides
- `operations/` - Operational procedures and troubleshooting
- `changelog/` - Change tracking and version history

## Documentation Standards

All documentation follows Markdown format and includes:
- Clear headings and structure
- Code examples where applicable
- Links to relevant resources
- Last updated timestamps
