# Product Overview

This is a Terraform-based Azure infrastructure project focused on security enhancements for cloud deployments. The project implements a hub-and-spoke network architecture with comprehensive security controls, monitoring, and automation capabilities.

## Key Features

- **Azure Infrastructure**: Complete Azure cloud infrastructure provisioning using Terraform
- **Security-First Design**: Integrated security scanning, policies, and compliance validation
- **Network Architecture**: Hub-and-spoke topology with bastion hosts, NSGs, and network segmentation
- **Identity & Access**: Key Vault integration, managed identities, and RBAC implementation
- **Monitoring & Logging**: Log Analytics workspace with comprehensive monitoring setup
- **Automation**: Azure Automation Account with DSC configurations for VM management

## Target Environment

- **Cloud Provider**: Microsoft Azure
- **Deployment Model**: Infrastructure as Code (IaC) using Terraform
- **Security Focus**: Enterprise-grade security controls and compliance validation
- **Workspace Support**: Multi-environment deployments (dev, staging, production)
